清除密码

搜索用户 → 清除密码 → 用户自行重设

1 搜索 / Search

2 选择用户 / Pick the user

    清除后该用户会被强制登出，并只能在游戏页的 「重设密码 / Reset password」 分页自行设置新密码。 管理员不会看到也不会决定新密码。

    After clearing, the user is signed out everywhere and can only set a new password themselves on the Reset password tab. You never see or choose it.
